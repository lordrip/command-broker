## Command broker
Simple http server that register incoming clients and allow to send commands and receive their results

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# watch mode
$ yarn start:debug

# production mode
$ yarn start:prod
```

## Test

```bash
# unit tests
$ yarn test

# e2e tests
$ yarn test:e2e

# test coverage
$ yarn test:cov
```
